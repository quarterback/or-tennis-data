Lineups — Ladder & Position Matrix

Two coach tools for the new state-tournament format. Pick a team by classification, district and gender. The Team Ladder shows that team's rank order of players ("letter order"), the court(s) each played and their win-loss record at each — a coach verifies the order and submits a final locked ladder before the playoffs. The Position Matrix shows where every rostered player appeared across the season, court by court, so it's easy to see that teams aren't stacking lower flights. Both read from real match results already in the system.

How the access code works
The first time a ladder is submitted for a team, the code you type claims that team — keep it safe and reuse it. After that, only someone with the same code can change or lock that team's ladder. Everyone can still view the ladder and matrix without a code. (Challenge-match results are not entered here — keep those on your paper form.)
